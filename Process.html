Adoption And Fostering Process


    To be able to adopt a dog from AORD shelter you need to register yourself first then fill up an adoption form which contains the user details such as firstname, lastname, address and more informations as well as the dog reference which the user want to adopt.

    Regarding the fostering process a person can foster a dog but not more than two for a period of two to three weeks, then if then person want to adopt the dog afterward, he will have to go through the same process of adoption.
